# Matching Game 🎯

A fun, interactive word association matching game built with React and Tailwind CSS!

## Features

- ✅ **Three Difficulty Levels**: Easy, Medium, and Hard
- 🎮 **Drag-and-Drop Gameplay**: Intuitive matching mechanics
- 🔊 **Sound Effects**: Audio feedback for correct/incorrect matches
- 🎨 **Beautiful UI**: Modern gradient design with smooth animations
- 📱 **Responsive Design**: Works on desktop and mobile
- 🖨️ **Printable Version**: Offline paper-based version included

## How to Play

1. Click on a difficulty level (EASY, MEDIUM, or HARD)
2. Drag items from the left column
3. Drop them onto their matching pairs on the right
4. Get immediate feedback with sounds and visual effects
5. Match all pairs to complete the level!

## Game Content

### Easy (10 pairs)
Basic everyday associations: Shoes↔Socks, Dog↔Bone, etc.

### Medium (10 pairs)
School and home-related pairs: Teacher↔Student, Library↔Books, etc.

### Hard (10 pairs)
Conceptual opposites and abstract relationships: Sun↔Moon, Buy↔Sell, etc.

## Playing Offline

The game includes a printable version you can use with paper or on a whiteboard!

## Technologies Used

- **React** - Component-based UI framework
- **Tailwind CSS** - Utility-first CSS styling
- **Framer Motion** - Smooth animations
- **GitHub Pages** - Free hosting

## Getting Started

### Online
Visit: [https://sarahmoreton.github.io/matching-game/](https://sarahmoreton.github.io/matching-game/)

### Local Development
```bash
git clone https://github.com/SarahMoreton/matching-game.git
cd matching-game
npm install
npm start
```

The app will open at `http://localhost:3000`
